 #  Career Path Explorer

  A modern React-based web application that helps computer engineering students explore different career paths and discover curated high-quality learning resources including YouTube playlists and trusted websites.

  This project demonstrates frontend development using React, dynamic UI rendering, and structured resource mapping for real-world career guidance.

  ---

  ##  Project Overview

  Choosing the right tech career can be confusing due to information overload and scattered learning resources.

  This project solves that problem by providing:

  - 25+ curated tech career paths
  - Step-by-step roadmaps for each domain
  - Handpicked YouTube channels
  - Trusted learning websites
  - Premium glassmorphism UI
  - Fast search functionality
  - Fully responsive modern layout

  Built as a frontend-focused project for deployment and real-world usability.
